# 本地 Benchmark 原始输出 / Local Benchmark Outputs

这个目录用于保存 `benchmarks/adapter-memory/run.mjs` 生成的原始 benchmark 输出。  
This directory stores raw benchmark outputs generated by `benchmarks/adapter-memory/run.mjs`.

raw files 可能包含本地路径、平台诊断信息以及 verbose Codex/Claude Code event streams，因此默认被 git 忽略。  
Raw files may include local paths, platform diagnostics, and verbose Codex/Claude Code event streams, so they are intentionally ignored by git.

面向维护者的脱敏摘要会生成到 `docs/benchmark-artifacts/results/`。  
Maintainer-facing sanitized summaries are generated under `docs/benchmark-artifacts/results/`.
